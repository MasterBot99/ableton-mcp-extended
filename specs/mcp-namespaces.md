# Lyra — MCP Namespaces

Erweiterte MCP-Tool-Oberfläche über Producer Pal:

```
lyra.memory.read          — read project memory, session history, user preferences
lyra.memory.write         — write memory (requires user confirmation for destructive ops)
lyra.arrangement.coach    — analyze arrangement density, suggest consolidations
lyra.rack.librarian       — list, tag, and recall Macro Variations across all racks
lyra.dummy.build          — generate dummy clip envelopes for a target parameter
lyra.comp.assist          — monitor comping takes, suggest best sections
lyra.sample.similar       — find similar samples using Live's built-in similarity engine
lyra.perform.audit        — audit Follow Actions, launch modes, and scene probability
lyra.perform.sequencer    — sequence Macro Variations or dummy clips in Arrangement
lyra.midi.coach           — recommend MIDI Transformations for selected clips
lyra.midi.mutate          — apply Max for Live MIDI Tools to generate or transform notes
lyra.latency.watchdog     — scan device chain latency, recommend flatten targets
lyra.push.proxy           — query and control Push 3 session state via OSC
```
